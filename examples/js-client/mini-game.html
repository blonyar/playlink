Playlink Mini Game Example

A tiny browser example that uses the JavaScript helper and sends movement events through Playlink room broadcasts.

  1. Start the Playlink server with rustup run stable cargo run.
  2. Connect, create a room, join it, then move with WASD or arrow keys.
  3. Click Add Bot for a one-page multiplayer check, or open another browser tab and join the same room.
